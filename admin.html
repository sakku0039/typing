クラス共有の流れ

サーバーなしで全員に同じ問題を配るための運用です。

  1. このページでJSONを編集、または外部AIで作成したJSONをインポートします。
  2. 「共有用 questions.json をダウンロード」を押します。
  3. GitHub Pagesで公開しているリポジトリの questions.json を同名ファイルで置き換えます。
  4. 生徒は通常の index.html を開くだけで、更新後の問題を読み込みます。

静的サイトのため、管理者ページからGitHub上のファイルを直接書き換えることはできません。

外部AI用プロンプト

このプロンプトをコピーし、外部AIでJSONを生成して、そのままインポートまたは questions.json として公開できます。
